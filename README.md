# MoodTodayWithTCA
- MoodToday에 이어서 Combine + SwiftUI + TCA 적용 및 학습을 위한 레포지토리입니다.

## Project Structure
- Data Layer: DTO, Repository 패턴 적용, APIs
- 도메인 (화면) 별로 구분

## Tech Stack
- Architecture Pattern: The Composable Architecture 
- Layout: SwiftUI
- Networking: Moya + Alamofire
- Test: XCTest
- Other Tools: Tuist(Not Yet)
